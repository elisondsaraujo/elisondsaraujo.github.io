# Elison D. Saraujo - Portfolio Website

Welcome to my personal portfolio website! This is a modern, responsive portfolio built with HTML, CSS, and JavaScript.

## 🌐 Features

- **Responsive Design** - Looks great on desktop, tablet, and mobile devices
- **Modern UI** - Clean and professional design with smooth animations
- **Smooth Navigation** - Easy-to-use navigation with smooth scrolling
- **Project Showcase** - Highlight your best projects
- **Contact Section** - Multiple ways to get in touch
- **Performance Optimized** - Fast loading and smooth interactions

## 📁 File Structure

```
├── index.html       # Main HTML file
├── style.css        # Styling and responsive design
├── script.js        # Interactive features and animations
└── README.md        # This file
```

## 🚀 Getting Started

### View Online
Visit: [https://elisondsaraujo.github.io](https://elisondsaraujo.github.io)

### Local Development
1. Clone this repository:
   ```bash
   git clone https://github.com/elisondsaraujo/elisondsaraujo.github.io.git
   ```

2. Open `index.html` in your browser

3. Make changes and refresh to see updates

## ✏️ Customization

### Update Personal Information
Edit `index.html` to change:
- Your name and title
- About me section
- Project descriptions
- Contact links (GitHub, LinkedIn, Email, etc.)

### Modify Styling
Edit `style.css` to customize:
- Colors (CSS variables at the top)
- Fonts and sizes
- Layout and spacing
- Animations

### Add More Projects
In the Projects section, duplicate and modify project cards:
```html
<div class="project-card">
    <div class="project-icon">
        <i class="fas fa-code"></i>
    </div>
    <h3>Your Project Name</h3>
    <p>Project description here</p>
    <a href="#" class="project-link">Learn More →</a>
</div>
```

## 🎨 Color Scheme

The portfolio uses a modern color scheme with two main colors:
- **Primary**: Indigo (#6366f1)
- **Secondary**: Purple (#8b5cf6)

Change these in `style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
}
```
